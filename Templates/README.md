# 🩹 SRE Roadmap: Templates

このディレクトリには、SRE の「責めない文化 (Blameless Culture)」を実践するためのテンプレートが含まれています。

## 📄 ファイル一覧

- **[POST_MORTEM_TEMP.md](./POST_MORTEM_TEMP.md)**:
  - 障害（インシデント）を「ギフト」に変えるためのポストモーテム・テンプレート。
  - 発生事象、タイムライン、根本原因（5 Whys）、再発防止策（Action Items）を網羅しています。

## 🛠 使い方

1. インシデントが解決した後、できるだけ記憶が新しいうちに `POST_MORTEM_TEMP.md` をコピーして記録を開始してください。
2. 目的は「誰かを責めること」ではなく「システムを改善すること」です。
3. 抽出された Action Items に優先順位をつけて実行し、同じ障害を二度と起こさない仕組みを作ります。
