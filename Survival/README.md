# 🎯 SRE Roadmap: Survival Strategy

このディレクトリには、シリーズ最終回「SRE 生存戦略：技術選定と市場価値」で紹介した、未経験から SRE として価値を出すための戦略的資産が含まれています。

## 📄 ファイル一覧

- **[Diagrams_Survival.md](./Diagrams_Survival.md)**:
  - **技術選定マトリクス**: 希少性と汎用性のバランスから、どの技術に投資すべきかを整理した図解。
  - **Vault アーキテクチャ図**: 記憶に頼らず、仕組み（外部脳）で実務経験の壁を越えるための構成図。

## 💡 生存戦略の 3 つのステップ

1. **基礎の徹底と自動化**: GUI を捨て、すべての設定をコード (IaC) で行う姿勢を持つ。
2. **貢献の可視化**: 学んだことや構築した仕組みを公開し、「信頼」という資産を築く。
3. **「技術」ではなく「仕組み」を売ること**: 特定のツールに依存せず、問題解決のプロセスそのものを価値として提供する。
