# 🚀 SRE Roadmap Vault 2026

このリポジトリは、YouTube シリーズ「[2026年版 SRE 独学ロードマップ](https://www.youtube.com/playlist?list=TEST_PLAYLIST_ID)」で使用されたすべての図解、テンプレート、コードサンプルを集約した「外部脳（Vault）」の実体です。

## 📂 コンテンツ一覧

### 1. 📊 ロードマップ & 図解 ([/Diagrams](./Diagrams))
- SRE の学習ステップ、各技術要素のつながりを視覚化した図解（Mermaid 形式および画像）。
- [Platform_Engineering.md](./Diagrams/Platform_Engineering.md): プラットフォーム・エンジニアリングの概念図。

### 2. 📝 SLO 設計ワークシート ([/Worksheets](./Worksheets))
- [SLO_Design_Template.md](./Worksheets/SLO_Design_Template.md): 信頼性を「指標」で管理するための具体的な 4 ステップ・ワークシート。

### 3. 🏗️ テンプレート集 ([/Templates](./Templates))
- **[IaC_Terraform_Structure](./Templates/IaC_Terraform_Structure)**: SRE 標準の Terraform ディレクトリ構成案。
- **[GitOps_GitHub_Actions](./Templates/GitOps_GitHub_Actions)**: GitHub Actions での CI/CD (OIDC) 設定サンプル。
- **[Postmortem_Template.md](./Templates/Postmortem_Template.md)**: 障害を「ギフト」に変え、組織を強化するための責めないポストモーテム・テンプレート。

### 4. 📈 オブザーバビリティ ([/Observability](./Observability))
- Prometheus / Grafana のセットアップコード (docker-compose) およびダッシュボード構成案。

### 5. 🛣️ Platform Engineering 実践 ([/Platform](./Platform))
- Backstage 等の IDP (Internal Development Platform) 導入に向けたチェックリスト。

### 6. 🎯 生存戦略 ([/Survival](./Survival))
- 技術選定マトリクス、33歳未経験から SRE として価値を出すための戦略図解。

## 🛠️ 使い方

1. このリポジトリを `git clone` するか、必要なファイルを直接コピーしてください。
2. 各ディレクトリの `README.md`（順次追加予定）に従って、自分の環境に合わせた「Vault」を構築してください。
3. 学習の記録は、自分なりの言葉で書き換えることが「静かな成長」への近道です。
