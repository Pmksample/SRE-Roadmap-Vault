# 📝 SRE Roadmap: Worksheets

このディレクトリには、SRE の実践を支援するためのワークシートが含まれています。

## 📄 ファイル一覧

- **[SLO_Design_Template.md](./SLO_Design_Template.md)**:
  - 信頼性を「指標」で管理するための 4 ステップ・ワークシート。
  - SLI の選定から SLO の設定、エラー予算の運用までを網羅しています。

## 🛠 使い方

1. `SLO_Design_Template.md` をコピーして、自分のプロジェクト用のドキュメントを作成してください。
2. チームメンバーと一緒に「何がユーザーの幸せに直結するか」を議論しながら埋めていくのがおすすめです。
